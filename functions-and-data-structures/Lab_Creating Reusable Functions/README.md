# Lab: Creating Reusable Functions
---
## Introduction
In this lab, you’ll build a small utility module with reusable Python functions for basic data cleaning and summarising. You’ll use core data structures (lists and dictionaries) and introduce Pandas as a lightweight tool inside functions to prepare analysis-ready data. You’ll validate outputs with assertions, refactor repetitive code, and version your work with Git and GitHub.

---
## Learning Objectives
- Define and reuse custom functions to modularise analytical code
- Manipulate Python data structures to manage and validate structured data
- Use Pandas inside functions for basic cleaning and summarising
- Validate outputs using assertions
- Refactor repetitive logic and version your work with Git
- Combine functions with lists and dictionaries to solve small, structured problems.
---
## Exercise Overview
You will work with a small structured dataset, clean and validate it using reusable functions, and prepare it for analysis.

---
## How to Work Through the Lab
1. Review the sample dataset provided in the notebook.
2. Write reusable functions for cleaning and summarising data.
3. Validate outputs with assertions.
4. Refactor code into reusable helpers.
5. Commit and push your work to GitHub.

A completed lab includes a documented Jupyter notebook demonstrating functions, lists, dictionaries, and iteration.
