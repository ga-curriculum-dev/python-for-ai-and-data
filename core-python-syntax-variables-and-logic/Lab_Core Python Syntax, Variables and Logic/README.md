# Lab: Core Python Syntax, Variables and Logic

## Introduction

In this lab, you’ll practise writing core Python code using variables, expressions, conditionals, and loops. You’ll work step by step in a Jupyter Notebook to apply logical thinking, document your code clearly, and version your work using Git and GitHub.

This lab reinforces the foundational programming concepts needed for data analytics and AI workflows.

---

## Learning Objectives

By the end of this lab, you’ll be able to:

- Declare and manipulate variables using core Python syntax  
- Apply conditionals to control program logic  
- Use loops to automate repetitive tasks  
- Write clear, readable, and well-documented Python code
- Trace code execution to predict outputs and identify basic logic errors.  
- Version your work using Git and GitHub  

---

## Exercise Overview

In this lab, you’ll create a Jupyter Notebook where you practise core Python concepts through guided exercises. You’ll write Python code using variables, conditionals, and loops, document your work using markdown cells, and organise your project in a reproducible structure.

---

## How to Work Through the Lab

Follow the steps below to complete the lab in order. Run each code cell as you go to confirm your code works as expected.

### 1. Create Your Notebook

- Launch Jupyter Notebook from Anaconda Navigator  
- Create a new notebook using **Python 3 (ipykernel)**  
- Rename the notebook to `core_python_logic.ipynb`  

---

### 2. Working with Variables and Expressions

Create a **code cell** and write Python code that:

- Assigns at least two variables  
- Performs a simple calculation using those variables  
- Displays the result  

Then create a **markdown cell** explaining what the code does.

---
### 3. Applying Conditionals

Add a new **code cell** where you:

- Create a variable that represents a value (for example, a score)  
- Use an `if / else` statement to make a decision based on that value  
- Print a message depending on the condition  

Add a **markdown cell** explaining the logic of your conditional.

---

### 4. Using Loops

Add another **code cell** where you:

- Use a `for` loop or a `while` loop  
- Print values as the loop runs  
- Modify a variable inside the loop  

Document what the loop is doing using a **markdown cell**.

---

### 5. Organise Your Project

- Create the folders `notebooks/` and `data/` if they do not already exist  
- Move your notebook into the `notebooks/` folder  
- Create a `README.md` file at the root of the project describing the purpose of this lab

A completed lab includes a documented Jupyter notebook demonstrating variable usage, calculations, and conditional logic.

### Tips for Success

- Run each notebook cell as you go to confirm your code works  
- Use markdown cells to clearly explain your logic  
- Choose meaningful variable names  
- Keep your project structure clean and organised
  
## Version Control and Submission

1. Open a terminal in your project folder.

2. Initialise a Git repository:

```bash
git init
git add .
git commit -m "Complete core Python syntax and logic lab"
```
3. Create a new repository on GitHub.

4. Connect your local project to GitHub and push your work:

```bash
git branch -M main
git push -u origin main
```
