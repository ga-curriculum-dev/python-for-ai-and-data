# Core Python Syntax, Variables and Logic — Instructor Guide

**Module Time:** 120 minutes

---

## What This Module is About

This module introduces learners to core Python programming concepts, focusing on syntax, variables, expressions, conditionals, and loops. It builds logical thinking skills required to write, debug, and reason about Python code used in data analytics and AI workflows.

---

## Core Objectives

- Help learners become comfortable writing and running Python code  
- Develop understanding of variables, data types, and expressions  
- Introduce control flow using conditionals and loops  
- Reinforce readable, well-documented coding practices  
- Prepare learners for more advanced data-focused Python modules  

---

## Must-Hit Topics

- Core Python syntax and data types  
- Variables and expressions  
- Conditional logic (`if / else`)  
- Looping constructs (`for`, `while`)  
- Debugging common syntax and logic errors  
- Versioning Python work with Git  

---

## 1. Python Syntax and Data Types
**(Time: ~25 minutes)**

- **Purpose:**  
  Introduce Python’s syntax and built-in data types as the foundation for all programming tasks.

- **Talking Points:**
  - Python’s emphasis on readability and indentation  
  - Common data types (`int`, `float`, `str`, `bool`)  
  - Assigning values to variables  
  - Common beginner syntax errors  

- **Teaching Tips:**
  - Live-demo simple variable assignments  
  - Encourage learners to experiment in small steps  
  - Reinforce that errors are part of the learning process  

---

## 2. Variables and Expressions
**(Time: ~25 minutes)**

- **Purpose:**  
  Help learners understand how variables store and transform values through expressions.

- **Talking Points:**
  - Naming variables clearly and consistently  
  - Performing arithmetic operations  
  - Reusing and updating variable values  
  - Interpreting outputs  

- **Teaching Tips:**
  - Ask learners to predict outputs before running code  
  - Emphasise readability over cleverness  
  - Encourage descriptive variable names  

---

## 3. Control Flow with Conditionals
**(Time: ~25 minutes)**

- **Purpose:**  
  Introduce decision-making logic using conditional statements.

- **Talking Points:**
  - Boolean logic and comparison operators  
  - `if / else` statements  
  - Real-world examples of decision logic  
  - Common mistakes with indentation and conditions  

- **Teaching Tips:**
  - Use simple, relatable examples (scores, thresholds, flags)  
  - Walk through code line by line  
  - Ask learners to explain the logic in plain English  

---

## 4. Control Flow with Loops
**(Time: ~25 minutes)**

- **Purpose:**  
  Show how loops automate repetitive tasks and reduce manual work.

- **Talking Points:**
  - When to use `for` loops vs. `while` loops  
  - Loop counters and iteration  
  - Modifying variables inside loops  
  - Avoiding infinite loops  

- **Teaching Tips:**
  - Visualise loops using simple sequences  
  - Encourage learners to stop and inspect outputs  
  - Reinforce loop structure and indentation  

---

## 5. Wrap-Up and Lab Overview
**(Time: ~20 minutes)**

- **Purpose:**  
  Consolidate learning and prepare learners for the guided lab.

- **Talking Points:**
  - How variables and control flow work together  
  - Importance of readable, well-documented code  
  - How today’s concepts connect to future data analysis tasks  
  - Overview of the Module 2 lab  

- **Teaching Tips:**
  - Ask learners to summarise key concepts aloud  
  - Reassure learners that repetition will reinforce understanding  
  - Clearly explain expectations for the lab  

---

## Wrap-Up Reflection
**(Time: ~5 minutes)**

- **Key Takeaways:**
  - Python logic is built from small, readable steps  
  - Conditionals and loops allow programs to make decisions and repeat tasks  
  - Clear code structure makes debugging and collaboration easier  

- **Next Steps:**  
  Learners move into the Module 2 lab to practise core Python syntax, logic, and control flow in a guided Jupyter Notebook exercise.
